# MyContacts-iOS
## Swift로 연락처 앱 만들기

## 프로젝트 소개

**PokeAPI**를 활용한 iOS 연락처 앱입니다.

서버와의 통신으로 랜덤 이미지를 받아 프로필 사진으로 지정하고, 코어데이터를 사용해서 기기 내에 이름과 전화번호를 저장/관리할 수 있습니다.

>**API**: https://pokeapi.co/
